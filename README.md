# ⚡ Energy Cost Calculator

A web app to calculate and track your household energy costs across different appliances. Compare costs across 30+ countries with automatic currency conversion.

## 🌟 Features

- **Multi-Country Support**: Pre-loaded electricity rates for 30+ countries
- **Automatic Currency Conversion**: Seamlessly switch between ₦, $, £, €, ¥, and more
- **Quantity Support**: Add multiple units of the same appliance at once
- **Real-Time Calculations**: Instant cost updates for daily, monthly, and yearly expenses
- **Smart Money-Saving Tips**: Personalized recommendations based on your usage
- **Minimal Design**: Clean, modern interface that's easy on the eyes
- **Mobile Responsive**: Works perfectly on phones, tablets, and desktops
- **No Backend Required**: Runs 100% in the browser - just open and use

## 🛠️ Installation

No installation needed! Just download and open `energy-calculator.html` in any modern web browser.

### Usage
1. Download `energy-calculator.html`
2. Double-click to open in your browser
3. Start adding appliances!

## 💡 How to Use

1. **Select Your Location**: Choose your country from the dropdown (or select Custom)
2. **Add Appliances**: 
   - Pick from common appliances or create custom ones
   - Set power consumption (Watts)
   - Set hours used per day
   - Set quantity (if you have multiple)
3. **View Costs**: See daily, monthly, and yearly costs automatically calculated
4. **Get Tips**: Receive personalized energy-saving recommendations
5. **Compare**: Switch countries to compare costs across locations

## 🔧 Technical Details

- **Tech Stack**: Pure HTML, CSS, JavaScript (Vanilla - no frameworks!)
- **File Size**: ~25KB (single file)
- **Browser Support**: All modern browsers (Chrome, Firefox, Safari, Edge)
- **No Dependencies**: Zero external libraries required
- **Data Storage**: All calculations done client-side, no data sent to servers
- **Privacy**: 100% private - all data stays in your browser
